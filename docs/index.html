Everything you need to steer Mac audio.

AudioRouter makes audio sources, outputs, and saved routes visible so you can make changes without digging through system panels.

AU

Menu bar control

Open a compact router, pick an app, choose an output, and adjust gain from one quick surface.

CA

Core Audio devices

See connected Bluetooth, built-in, USB, HDMI, AirPlay, and aggregate devices with clear status.

PT

Process taps

Use modern macOS audio capture where available, with honest labels when a route needs more backend work.

GP

Group Play

Create visual speaker groups, test outputs, and save preferred multi-speaker destinations.
