Simple routing flow

  1. Open AudioRouter from the menu bar.
  2. Pick a source app in Routes.
  3. Select an output device or Group Play target.
  4. Approve macOS audio capture when prompted.
  5. Use Command-= or Command-- to adjust the selected source by 1%.

Important routing note

Some app-to-device routes are experimental because macOS does not expose every SoundSource-style routing feature through public APIs alone. AudioRouter saves the route and shows the real status instead of pretending unsupported routing is live.
