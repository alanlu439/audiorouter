Run the app safely.

The current package may be unnotarized, so macOS can show a verification warning. Use the manual flow below and keep Gatekeeper enabled.

Download AudioRouter-macOS.zip Preview the PDF user manual
  1. Open the ZIP and read AudioRouter-User-Manual.pdf.
  2. Move AudioRouter.app to Applications.
  3. Control-click AudioRouter.app and choose Open.
  4. Approve macOS audio capture and automation prompts when needed.

If you need a fully notarized installer, wait for a future Developer ID release. Do not disable SIP or Gatekeeper.
